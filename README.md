## 1. 简介

&emsp;&emsp;记载工程实践问题的解决策略与关键要点，分享各种实用案例，追踪前沿技术发展，囊括 AI 全栈知识，涵盖大模型、编程技术、机器学习、深度学习、强化学习、图神经网络、语音识别、NLP 及图像识别等领域。 
 



1. 代码结构和内容简介

    ```
    ai_wiki (AI全栈教学知识，以Markdown, Jupyter Notebook汇总知识体系)

    │   ├── 系统：Windows/Linux
    ├── 02_程序代码 
    │   ├── 编程：python, c, c++, 数据库, LeetCode
    │   ├── 实战：常用工具、常见问题汇总
    ├── 03_数学基础（程序员必备数学知识）
    ├── 04_算法原理（传统算法，优化算法，遗传算法）
    ├── 05_机器学习（资源+原理+实战）
    ```
