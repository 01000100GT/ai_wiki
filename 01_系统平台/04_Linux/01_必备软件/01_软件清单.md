# 2. 小工具
## 2.1 Tmux

保持程序在后台运行的工具。

- 可以建立多个会话
- 每个会话下又可以建立多个子会话

1. 安装

    ```shell
    sudo apt install tmux
    ```

2. 命令

   - 启动新窗口:  tmux new -s xxx    # xxx为名称
   - 打开已有会话:  tmux a -t xxx  # xxx为名称
   - 再会话中新建一个窗口: Ctrl + B之后，再按c
   - 切换窗口: Ctrl + B之后，再按窗口对应的数字
